获取和修改数据库的基本命令（注意命令中的分号！）

1.连接到MySql数据库（必要）：mysql -u root -p

2.查看数据：show databases;

3.创建数据库：create database <dbname>;（创建名为dbname的数据库）

4.使用创建的数据：use <dbname>;（使用名为dbname的数据库）

5.查看当前使用的数据库中有哪些表：show tables from <dbname>;

6.初始化数据：source 路径\*.sql(使用sql脚本初始化数据)

注意

1.若sql脚本中的数据量过大，无法直接打开，需要使用source命令进行初始化

2.路径中不能包含中文字符和奇怪的标点符号(error:22)

7.删除数据库：drop database <dbname>（删除名为dbname的数据库）

其他操作

desc <tablename>：查看名为tablename的表的结构

select database();：查询当前使用的数据库

select version();：查看当前MySql版本号

\c：命令，结束一条语句
